# Syntropy MVP

Minimal LibGDX prototype for a possession-based colony sim.

## Controls

- `P`: Possess/unpossess nearest colonist to mouse cursor
- `WASD`: Move the possessed colonist

## What the MVP does

- Renders a 10x10 grid with a miner and a stockpile.
- Colonist wanders, hauls stone from the miner to the stockpile.
- Hunger slowly decays; colonist seeks food at the stockpile.

## Run

```powershell
./gradlew run
```
